# Prompt-Powered Kickstart: Django Motivational Quotes App

## 📌 Objective
This project demonstrates how to build a simple Django web application using AI-assisted learning. The goal is to create a beginner-friendly motivational quotes app and document the process clearly.

---

## ⚙️ Technology Used
I used Django, a Python web framework used to build secure and scalable web applications quickly. It is commonly used in backend development.

**Real-world example:** Instagram

---

## 💻 System Requirements
- Python 3+
- pip
- VS Code
- Web browser

---

## 🚀 Setup Instructions

```bash
pip install django
django-admin startproject quote_project
cd quote_project
python manage.py startapp quotes
python manage.py runserver
