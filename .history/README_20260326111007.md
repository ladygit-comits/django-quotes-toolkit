Prompt-Powered Kickstart: Django Motivational Quotes App
📌 Objective

This project demonstrates how to build a simple Django web application using AI-assisted learning.
The goal is to create a beginner-friendly motivational quotes app that displays a random quote each time the page is refreshed and documents the development process clearly.

⚙️ Technology Used

Django - a Python web framework used to build secure and scalable web applications quickly.
It is widely used for backend development.

Real-world example: Instagram (uses Django for backend features like feeds and posts)

💻 System Requirements
Python 3+
pip
VS Code (or any code editor)
Web browser
🚀 Setup Instructions
Install Django:
pip install django
Create Django project:
django-admin startproject quote_project
cd quote_project
Create app:
python manage.py startapp quotes
Run development server:
python manage.py runserver

Open http://127.0.0.1:8000/ in your browser to see the app.

📝 Minimal Working Example

quotes/views.py

from django.shortcuts import render
import random

# Handles homepage requests
def home(request):
    # List of motivational quotes
    quotes = [
        {"text": "Stay consistent.", "author": "You"},
        {"text": "Keep going.", "author": "Unknown"},
        {"text": "You’ve got this.", "author": "Motivation"},
    ]

    # Pick a random quote
    quote = random.choice(quotes)

    # Render template and pass quote
    return render(request, 'home.html', {'quote': quote})

quotes/templates/home.html

⚠️ Common Issues & Fixes
ModuleNotFoundError: No module named 'django'
Fix: pip install django
Server doesn’t reload after changes
Fix: Stop server (Ctrl+C) and run python manage.py runserver again
Template not found
Fix: Make sure quotes/templates/ exists and template path matches
